# Dashboard Overview

The AOV Bundle dashboard is your central hub for managing bundle offers, tracking performance, and accessing key features.

<Callout type="info">
  **Prerequisites:** AOV Bundle must be installed on your Shopify store.
</Callout>

## Overview

When you open AOV Bundle, the dashboard displays a personalized greeting, quick start tasks, performance metrics, and helpful resources. The left sidebar provides navigation to all app sections: Dashboard, Offers, Widget Display, Free Gift, Analytics, and more.

## Sections

<Steps>

### Step 1: Review the navigation sidebar

The left sidebar contains the main navigation menu. Click any item to navigate between sections:

- **Dashboard** — Home overview
- **Offers** — Create and manage your bundle offers
- **Widget Display** — Customize how offers appear on your store
- **Free Gift** — Set up free gift campaigns
- **Analytics** — Track offer performance

![Navigation sidebar](/images/dashboard-overview/step-1.png)

### Step 2: Check the Quick Start widget

The **Quick Start** section shows your current offer count and provides shortcuts to create your first offers. Complete the suggested tasks to get up and running quickly.

![Quick Start widget](/images/dashboard-overview/step-2.png)

### Step 3: View the Dashboard Report

Below Quick Start, the **Dashboard Report** displays key performance metrics including revenue generated, additional revenue from bundles, and active campaign data.

![Dashboard Report](/images/dashboard-overview/step-3.png)

### Step 4: Access the Help Center

At the bottom of the dashboard, the **Help Center** widget provides quick links to documentation, support resources, and feature guides.

![Help Center](/images/dashboard-overview/step-4.png)

### Step 5: Choose your language

Click the **Choose Language** button at the top right of the page to change the app interface language.

![Choose Language button](/images/dashboard-overview/step-5.png)

</Steps>

<Callout type="warning">
  Dashboard metrics may take a few minutes to update after creating or modifying offers. Refresh the page to see the latest data.
</Callout>
