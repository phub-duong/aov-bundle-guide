# Track Offer Performance

Drill into individual offer analytics to understand which bundle campaigns are driving the most revenue. Compare performance across time periods and identify optimization opportunities.

> **Info:** **Prerequisites:** AOV Bundle installed with active offers that have generated orders.

## Overview

You can access offer-specific analytics in two ways: from the offer list (click the analytics icon) or from the Analytics page (filter by a specific offer).

## Steps

### Step 1: Navigate to offer analytics

From the **Offers** page, click the **Analytics** icon (chart icon) in the Actions column next to the offer you want to analyze. This opens the analytics page filtered to that specific offer.

![Click analytics icon](/.gitbook/assets/offer-performance/step-1.png)

### Step 2: Review offer-specific metrics

The analytics page now shows data only for the selected offer. Review the revenue chart, conversion metrics, and trend indicators specific to this campaign.

![Offer-specific metrics](/.gitbook/assets/offer-performance/step-2.png)

### Step 3: Compare time periods

Use the date picker to select different time periods and compare performance. The trend indicators show percentage changes compared to the previous equivalent period (e.g., this week vs. last week).

![Compare time periods](/.gitbook/assets/offer-performance/step-3.png)

### Step 4: Review the data table

Scroll down to the detailed data table that breaks down metrics by day. This table shows exact values for revenue, orders, and additional metrics for each date in your selected range.

![Data table](/.gitbook/assets/offer-performance/step-4.png)

### Step 5: Switch to another offer

Use the **Offer filter** dropdown at the top to switch to a different offer's analytics, or select **All Offers** to return to the aggregate view.

![Switch offers](/.gitbook/assets/offer-performance/step-5.png)


> **Warning:** Offers in **Draft** status do not generate analytics data. Make sure to activate an offer before expecting to see performance metrics.
