# Widget Design Overview

AOV Bundle provides dedicated widget designs for each offer type. Customize colors, layouts, fonts, and content to match your store's branding.

> **Info:** **Prerequisites:** AOV Bundle installed on your Shopify store.

## Overview

Navigate to **Widget Display** in the left sidebar to access the design editor. You will see design type cards for each offer widget. Each design type has its own set of customization options.

![Design type selection page](/.gitbook/assets/design-overview/step-1.png)

## Available Design Types

### Volume Table

Customize the volume discount pricing table that appears on product pages. Choose between **vertical** and **horizontal** layouts. Supports gradient backgrounds and custom color schemes.

### Frequently Bought Together (FBT)

Customize the FBT recommendation widget. Configure product card styling, "Add to Cart" button appearance, and pricing display.

### Mix-and-Match Widget

Customize the mix-and-match selection interface. Configure option group styling, product cards, and bundle summary display.

### Bundle Builder

Customize the bundle builder page layout. Configure category headers, product grids, and the bundle progress indicator.

### Child Product List

Customize how individual bundle products are displayed. Choose between **Classic** and **Minimal** layout styles.

## Pre-built Themes

Each design type includes 8 pre-built color themes you can apply with one click:

| Theme | Style |
|-------|-------|
| Basic | Neutral, minimal colors |
| Bloody Mary | Red and pink gradient |
| Forest | Green gradient |
| Ocean Blue | Blue gradient |
| Spring | Light pastel tones |
| Twilight | Purple gradient |
| Summer | Orange and yellow |
| Eastern Blue | Teal gradient |
