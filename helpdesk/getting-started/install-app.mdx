# Install AOV Bundle

Install the AOV.ai Bundle app to start creating bundle offers, volume discounts, and upsell campaigns that increase your store's average order value.

<Callout type="info">
  **Prerequisites:** You need a Shopify store with an active plan and admin access to install apps.
</Callout>

## Steps

<Steps>

### Step 1: Open the Shopify App Store

Go to the [Shopify App Store](https://apps.shopify.com/) in your browser, or navigate to **Apps** in your Shopify admin sidebar and click **Shopify App Store** at the bottom of the page.

![Shopify admin sidebar with Apps](/images/install-app/step-1.png)

### Step 2: Search for AOV Bundle

In the App Store search bar, type **"AOV.ai Bundle"** and press Enter. Click on the **AOV.ai Bundle** app listing in the search results.

![Search for AOV Bundle](/images/install-app/step-2.png)

### Step 3: Click Add app

On the app listing page, click the **Add app** button to begin the installation process.

![Click Add app](/images/install-app/step-3.png)

### Step 4: Review and install

Review the permissions that AOV Bundle requires, then click **Install app** to confirm the installation.

![Approve and install the app](/images/install-app/step-4.png)

### Step 5: Verify installation

After installation, you will be redirected to the AOV Bundle dashboard. You should see the welcome greeting **"Hi there! Welcome to AOV.ai Bundle"** along with the Quick Start widget.

![AOV Bundle dashboard](/images/install-app/step-5.png)

</Steps>

<Callout type="warning">
  If the app does not appear after installation, try refreshing your Shopify admin page. The app will appear under **Apps** in your Shopify admin sidebar.
</Callout>
